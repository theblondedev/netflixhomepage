# Netflix Homepage Clone

This project is a recreation of the Netflix homepage built using **HTML and CSS**. The goal of the project was to practice front-end layout techniques, styling, and recreating a real-world interface.

## Project Overview

The page replicates the general layout and design style of the Netflix landing page, including the hero section, navigation bar, and content rows. It focuses on building a structured layout and styling elements to closely resemble the original design.

## Technologies Used

* HTML5
* CSS3
* Flexbox
* JavaScript (basic interactivity)
* Basic responsive layout techniques

## Features

* Navigation bar layout
* Hero banner section
* Content rows styled similar to Netflix
* Basic responsive structure

## Current Status

This project is **not fully complete**. Some layout formatting still needs refinement and additional styling work. Due to time constraints during the assignment, the focus was primarily on building the main structure and core styling.

## What I Learned

* Structuring a webpage with semantic HTML
* Using CSS to recreate real-world layouts
* Working with Flexbox for positioning elements
* Managing spacing, alignment, and styling consistency

## Future Improvements

If I revisit this project, I would like to:

* Improve responsive behaviour across screen sizes
* Refine formatting and layout alignment
* Add additional styling to better match the original design
* Potentially add JavaScript for interactive elements
